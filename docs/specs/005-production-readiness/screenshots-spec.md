## Screens to Capture

### 1. Home Screen (Accueil)
- Shows the main dashboard with election overview
- Candidate cards visible
- Theme highlights
- **Purpose**: First impression, shows the app's civic engagement focus

### 2. Survey Question
- Active survey question with answer options
- Progress indicator visible
- **Purpose**: Shows the interactive questionnaire feature

### 3. Candidate List (Candidats tab)
- Grid/list of all candidates with photos and party names
- **Purpose**: Shows the candidate comparison feature

### 4. Assistant Chat
- Active conversation in "Comprendre" (Learn) mode
- At least 2 message exchanges visible
- Source citations visible in assistant response
- **Purpose**: Shows the AI-powered neutral assistant

### 5. Candidate Profile
- Individual candidate detail page
- Position cards with source links
- **Purpose**: Shows the depth of sourced information

### 6. (Optional) Comparison View
- Side-by-side candidate comparison on a theme
- **Purpose**: Shows the comparison feature

## Screenshot Guidelines

- Use French language in all screenshots
- Ensure the status bar shows a reasonable time and full battery
- Use realistic data (the bundled Paris 2026 dataset)
- Do not include any personal data or real user conversations
- Capture on both light and dark mode if supported
- Actual screenshots will be taken by the app operator after production build
