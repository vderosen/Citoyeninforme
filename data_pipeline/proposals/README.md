# Proposals

This folder contains the V1 card content (propositions) for the survey.

## Files
- `V1 of cards for the app - renumbered.xlsx`: The original Excel file provided with the canonical descriptions, titles, and candidates' stances.
- `proposals.json`: The JSON export of the Excel file, making it easier to parse and track programmatically.

These sources are compiled and injected into `src/data/elections/paris-2026/survey.json` via the data pipeline.
